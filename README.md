# 🚀 Optemiz v2.1.0

**Windows için Tam Otomatik Sistem Bakım ve Optimizasyon Aracı**

Optemiz, bilgisayarınızı tek tıkla temizleyen, hızlandıran, gizliliğini koruyan ve performansını artıran güçlü bir PowerShell aracıdır.

---

## ✨ Ana Özellikler

- **Ultra Otomatik Bakım** → Hiç soru sormadan komple bakım
- **Detaylı Öncesi-Sonrası Raporu** → RAM, Disk ve CPU iyileşmesi
- **Modern HTML Log Raporu** → Tarayıcıda güzel görünüm
- **Registry Yedekleme Sistemi** → Güvenli değişiklikler
- **Driver Güncelleme Kontrolü**
- **Kapsamlı Gizlilik Ayarları**
- **Oyun Optimizasyonu** (Game Mode + Yüksek Performans)
- **BSOD Analizi**

---

## 🚀 Hızlı Kullanım

1. `Start-Optemiz.bat` dosyasına **sağ tık → Yönetici olarak çalıştır**
2. Menüden **12** numarayı seçin → **Ultra Otomatik Bakım** (En önerilen)

### Diğer Önerilen Kullanımlar
- **6** → Oyun öncesi optimizasyon
- **10** → BSOD sorunu varsa
- **14** → Son bakım raporunu göster
